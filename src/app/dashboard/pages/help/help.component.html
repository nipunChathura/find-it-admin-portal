How the Find It Admin Portal works

Overview

The Find It Admin Portal is the central place to manage your platform: admin users, customers, products (items), merchants, categories, outlets, and payments. You must be logged in to use any section.

Sidebar (left): Use the menu on the left to open each section. Some items (e.g. Item, Outlet) expand to show sub-pages. The current page is highlighted.

Top toolbar: The bar at the top includes the app title, your username, Messages, Notifications (bell), Account (profile), and Logout. Use the menu icon to collapse or expand the sidebar.

dashboard Dashboard

What it is

The Dashboard is the main container. When you click Dashboard in the sidebar (or open the portal), you are taken to the default start page, which is Manage User.

What you can do

  • Use the sidebar to switch between User, Customer, Item, Merchant, Category, Payment, Notifications, Outlet, and Help.
  • The main content area (centre) shows the page for the menu item you selected.
 people User

What it is

Manage User lets you view and manage admin users who can log in to this portal. You can add new admins, edit their details, change status, approve pending users, or remove users.

Page layout

  • Top: Page title “Manage User” and an Add User button.
  • Filters: A Status dropdown (All, ACTIVE, INACTIVE, PENDING, APPROVED, FORGOT_PASSWORD_PENDING) and a Search box. Click Search to apply; Clear resets filters and reloads the list.
  • Table: Columns show ID, Username, Email, Status, Created Date, and Actions (Edit, Delete, and for system admins, Approve). You can sort by clicking column headers and change page size with the paginator at the bottom.

How to add a user

  1. Click Add User.
  2. In the dialog, enter Name, Username, Email, Password, and optionally Status (if your role allows).
  3. Click Save. The new user appears in the table and can log in with the username and password you set.

How to edit a user

  1. Find the user in the table and click the Edit (pencil) icon in the Actions column.
  2. Update Username, Email, or Status in the dialog. Only users with the right role can change status.
  3. Click Update to save.

Other actions

  • Delete: Click the delete (trash) icon, confirm in the dialog. The user’s status is set to deleted; they can no longer log in.
  • Approve: If you are a System Admin, you will see an Approve (check) icon for users with status PENDING. Click it to approve them. System Admin users themselves cannot be edited or deleted from this screen.
 person Customer

What it is

Manage Customer shows all customers (end users of the Find It platform). Customers are loaded from the API and can be filtered, searched, edited, or deleted.

Page layout

  • Filters: Status (All, ACTIVE, INACTIVE, PENDING), Membership (All, SILVER, GOLD, PLATINUM), and Search. Click Search to apply; Clear resets and reloads.
  • Table: Columns are ID, Name, Email, Phone, Membership, Status, and Actions (Edit, Delete). Sort by column header; use the paginator for page size and navigation.

How to edit a customer

  1. Click the Edit (pencil) icon on the row.
  2. In the dialog, update First name, Last name, Email, Phone number, NIC, Date of birth, Gender, Country, Membership, or Status.
  3. Click Update. Changes are sent to the backend.

How to delete a customer

Click the Delete (trash) icon, then confirm in the dialog. The customer is removed from the list (backend delete).

 inventory_2 Item

What it is

The Item menu has two sub-pages: Item (product/service catalogue) and Item Discount (discounts applied to items).

Item (catalogue)

  • View all items in a table. Use Search and Status / Availability filters to find items.
  • Add new items, Edit (pencil) or Delete (trash) from the Actions column. Sort and paginate as needed.
  • Items are typically linked to categories and merchants and can be used in offers and discounts.

Item Discount

  • Manage discounts that apply to specific items. Filter by Status and Item.
  • Use Add to create a discount (e.g. name, type, value, item, validity). Edit or Delete from the Actions column.
 store Merchant

What it is

Merchant is for managing merchants (businesses) that use the Find It platform. Merchants usually have one or more outlets and sell items. This page lists all merchants and lets you add, edit, or delete them.

What you can do

  • View the merchant list (often with search and filters).
  • Add Merchant: Create a new merchant (name, contact details, etc. as per the form).
  • Edit: Change merchant details from the Actions column.
  • Delete: Remove a merchant (confirm in the dialog). Deleting a merchant may affect linked outlets and items, depending on backend rules.
 category Category

What it is

Category lets you manage categories used to group items (e.g. Food, Electronics). Categories help customers and admins find and filter items.

What you can do

  • View all categories in a table (with search/filters if available).
  • Add Category: Create a new category (e.g. name, description).
  • Edit: Update a category’s name or other fields.
  • Delete: Remove a category. Ensure no items depend on it, or reassign them first, to avoid errors.
 payment Payment

What it is

Payment is for managing payment methods or payment configurations linked to outlets. You can open it from the main menu (Payment) or from under Outlet → Payment.

What you can do

  • View payments in a table, often filterable by Outlet and Status.
  • Add: Create a new payment method/configuration for an outlet (e.g. type, account details, status).
  • Edit: Change payment details or status from the Actions column.
  • Delete: Remove a payment configuration after confirming in the dialog.
 notifications Notifications

What it is

Notifications keep you updated on events (e.g. new signups, system alerts). You can see them in a dropdown from the bell icon in the toolbar, or on the full Notifications page.

Bell icon (toolbar)

  • Click the bell in the top toolbar to open a dropdown panel.
  • A red badge on the bell shows the number of unread notifications (hidden when zero).
  • In the dropdown you see: notification title, short message, time (e.g. “5m ago”), and type (info, success, warning, error). Unread items are highlighted.
  • Use the checkmark on a row to mark that notification as read, or click Mark all as read at the top.
  • Click View All to open the full Notifications page. The list is refreshed automatically every 10 seconds.

Notifications page (View All)

  • Open from the bell dropdown (View All) or from the sidebar (Notifications).
  • Shows all notifications in a list. You can Mark all as read or mark individual items as read. Same data as the dropdown, in a full-page view.
 storefront Outlet

What it is

Outlet represents a physical or logical location (e.g. a store branch). Under the Outlet menu there are three sub-pages: Outlet, Outlet Schedule, and Payment.

Outlet (list)

  • View all outlets. Add new outlets, edit (name, address, merchant, etc.), or delete. Use search and filters to find specific outlets.

Outlet Schedule

  • Configure opening hours and schedules for each outlet (e.g. Mon–Fri 9–5, Sat 10–2). Add, edit, or remove schedule entries so customers see when outlets are open.

Payment

  • Same as the main Payment menu: manage payment methods/configurations for outlets.

Tips

  • Sidebar: Use the menu icon (top-left of the toolbar) to collapse or expand the sidebar and get more space for the content.
  • Finding data: On list pages (User, Customer, Item, etc.), use Search and filters (Status, dropdowns) then click Search or press Enter. Use Clear to reset.
  • Sorting and paging: Click column headers to sort. Use the paginator at the bottom to change page size (e.g. 5, 10, 25, 50) and move between pages.
  • Account: Click your Account (profile) icon in the toolbar to open your profile and update your details or profile image.
  • Logout: Always use Logout when you finish your session to keep your account secure.
